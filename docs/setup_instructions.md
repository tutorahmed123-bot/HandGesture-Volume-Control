# Setup Instructions

## Quick Start
1. Install Python 3.8 or higher
2. Clone this repository
3. Run \pip install -r requirements.txt\
4. Run \python run.py\

## Detailed Setup

## Gesture Guide
- Show **OPEN HAND** to increase volume bar
- Show **CLOSED HAND** (fist) to decrease volume bar  
- Hold gesture for smooth control
- Works best with good lighting and steady hand
